## 项目概述
基于PRD文档开发的驾校场地智能监管系统，实现学员/教练/车辆统一管理，AI违规行为监测，实时预警告警等功能。

## 技术架构
- **前端**: Vue3 + Element Plus + H5移动端
- **后端**: SpringBoot + MyBatis-Plus + SpringSecurity  
- **AI能力**: 百度人脸识别 + 阿里云车牌识别 + OpenCV自定义模型
- **数据库**: MySQL 8.0 + Redis 6.0
- **部署**: Linux + Nginx

## 功能模块
1. 基础信息管理（学员/教练/车辆/场地）
2. 身份准入与核验（白名单机制）
3. 违规行为智能监测（核心功能）
4. 实时预警与告警
5. 违规记录与追溯管理
6. 统计报表模块

## 🚀 功能特性

### 核心功能模块
- **基础信息管理**：学员、教练、车辆、场地信息管理
- **身份核验系统**：人脸识别、车牌识别准入验证
- **违规行为监测**：实时视频监控、违规行为智能识别
- **预警告警系统**：实时预警推送、多级预警处理
- **记录追溯管理**：违规记录查询、证据查看、处理追踪
- **统计报表分析**：多维度数据统计、可视化图表展示
- **系统权限管理**：三级权限控制（管理员/场控负责人/普通操作员）

### 技术特色
- ✅ 响应式设计，支持PC端和移动端
- ✅ 实时摄像头调用和图像识别
- ✅ 动态权限控制和菜单过滤
- ✅ 实时数据更新和预警推送
- ✅ 完整的表单验证和交互反馈

## 🛠️ 技术栈

- **前端框架**：Vue 3 + Composition API
- **UI组件库**：Element Plus
- **状态管理**：Pinia
- **路由管理**：Vue Router 4
- **构建工具**：Vite
- **图表库**：ECharts
- **HTTP客户端**：Axios
- **工具库**：Day.js

## 🚀 部署建议

### 生产环境部署
```bash
# 构建项目
npm run build

# 部署到服务器
# 推荐使用Nginx作为反向代理服务器
```

### Docker部署（推荐）
```dockerfile
FROM node:18-alpine as build
WORKDIR /app
COPY package*.json ./
RUN npm ci --only=production
COPY . .
RUN npm run build

FROM nginx:alpine
COPY --from=build /app/dist /usr/share/nginx/html
EXPOSE 80
CMD ["nginx", "-g", "daemon off;"]
```
